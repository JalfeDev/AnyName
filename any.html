Ordered List: Priorities in Push (Local to Global)
  1. git add
  2. git commit "message"
  3. git status
  4. git push

Other forms of Commit & Push
  • Instead of the previous one
  • By adding and committing at the same time
  • use git commit -am "message"
  • git push
  • or use for branches: git push origin [branch]
Merge conflicts
  • It happens when 2 changes are made in the same file
  • Happens in groups, but sometimes also alone, like me
Git branch change
  • Tracking won't be linear, There are many branches storing files
  • The main one is "main"
  • You can check all branches with: git branch
  • Create and switch to a new branch with: git checkout -b [new branch]
  • Or select one created with: git checkout [new branch]
